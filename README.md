# Học Toán Lớp 8

## Project Overview
"Học Toán Lớp 8" is an educational web application designed to help eighth-grade students learn and practice mathematics through interactive lessons and exercises. The project utilizes modern web technologies to create a user-friendly experience for students to grasp mathematical concepts, such as algebra, geometry, arithmetic, and statistics.

## Installation
To get started with this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd học-toán-lớp-8
   ```

2. **Open the project:**
   Open `index.html` or `index_restructured.html` in your favorite web browser.

3. **Dependencies:**
   The app loads necessary dependencies via CDN links within the HTML files, so no additional installation is required for these dependencies.

## Usage
Once you have the project running, here's how you can navigate and utilize its features:

- **Lessons:** Explore various mathematical topics such as Algebra, Geometry, Arithmetic, and Statistics.
- **Exercises:** Engage with tasks tailored to each topic, complete with solutions available upon request.
- **Math Tool:** Utilize the built-in calculator for solving math expressions directly.
- **Quiz:** Take practice tests with multiple-choice questions and get immediate feedback on your answers.

## Features
- Interactive lessons on key eighth-grade mathematics topics.
- A variety of exercises with options to reveal solutions.
- A built-in calculator for user convenience.
- A quiz section with detailed answers and explanations to encourage learning through assessment.
- Responsive design for a seamless experience on both desktop and mobile devices.

## Dependencies
This project uses the following external libraries:
- Tailwind CSS (via CDN)
- Font Awesome (for icons via CDN)
- Function Plot (for graph plotting via CDN)

## Project Structure
The project is organized as follows:
```
/học-toán-lớp-8

└── index_restructured.html # Alternate version of the main HTML with additional features
```

- **index_restructured.html**: An alternative version of the application that may contain additional functionality or features such as quizzes and collapsible content for easier navigation.
